This project aims to develop machine learning models to predict individuals’ monthly income and classify them into income brackets based on demographic and financial-related features. 
Multiple regression and classification algorithms were applied to identify the most accurate and reliable models for both prediction and categorization tasks.
The results of this project can support data-driven decision-making in areas such as financial assessment, socioeconomic analysis, 
and targeted policy or business strategies by providing insights into income prediction and classification.
