# Income Prediction and Classification Model

## Overview

This project focuses on building machine learning models to predict individuals' monthly income and classify them into income brackets using demographic and financial data.

## Objectives

* Predict income using regression models
* Classify individuals into income categories
* Compare model performance for accuracy and reliability

## Tools & Technologies

* Python
* Pandas
* Scikit-learn
* Google Colab

## Key Features

* Data preprocessing and cleaning
* Feature analysis and selection
* Model training (Regression & Classification)
* Performance evaluation

## Impact

This project demonstrates how data-driven models can support decision-making in financial analysis, socioeconomic research, and policy development.
